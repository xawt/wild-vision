# wild-vision
PyTorch Lightning project template for wild/nature image recognition

## Installation

Install [uv](https://docs.astral.sh/uv/getting-started/installation/), then run:

```bash
uv sync
```

uv will download the required Python version and all dependencies automatically.
On Windows, PyTorch is installed with CUDA 13.0 support; on macOS/Linux the standard build is used.

## Running

```bash
uv run main.py
```
